How to Add Custom Sounds to Rocket League — Goals, Saves, Demos & Crossbars

Bakkboard lets you assign any audio file to in-game events — goals, saves, demolitions and crossbars. Every time the event fires, your sound plays. No injection, no mods, no risk.

Setting up custom sounds for in-game events in Bakkboard's Sounds panel.

What custom sounds do

Each time a supported event happens during a match, Bakkboard intercepts it from Rocket League's Stats API and plays the audio file you assigned to that category. The game audio is unchanged — Bakkboard's sounds play on top, through your speakers or headset, from a completely separate audio process.

Supported event categories:

  • Goal — fires when any qualifying goal is scored
  • Save — fires on saves and epic saves
  • Demolition — fires when a demolition occurs
  • Crossbar — fires when the ball hits the post or crossbar

Adding your audio files

Open Bakkboard and go to the Sounds tab. Each event category has its own drag-and-drop zone. Drag any audio file (MP3, WAV, OGG, M4A) into the zone for the category you want, or click the zone to open a file picker. The file is copied into Bakkboard's sounds folder and is immediately available.

You can add multiple files per category. Bakkboard will pick one at random each time the event fires — or play them in the order you set, if you prefer sequential mode.

Trim, fade and volume

Each file has an individual volume slider and a built-in trim editor. Open the trim editor by clicking the scissors icon next to any file. Drag the start and end handles on the waveform to cut the clip to exactly the length you want, with optional fade in and fade out. A global volume slider applies on top of all individual levels.

Who triggers the sound

For each category you can set the scope:

  • Me — only when you personally score, save, demolish or hit the post
  • My team — when anyone on your team triggers the event
  • Everyone — every event in the match, regardless of team

Setting Demolition to "Me" means your sound only plays when you demolish someone — not when you get demolished.

No injection, no ban risk

The sounds are played entirely by Bakkboard's own audio engine. Rocket League's process is never touched. Easy Anti-Cheat does not see Bakkboard because there is nothing to intercept — the app reads match events from the Stats API (a standard OS-level interface the game exposes to the operating system) and plays audio in a separate process, the same way any music player running alongside a game would.
