How to Load Workshop Maps in Rocket League (2026)

Bakkboard's Workshop tab lets you browse, search and load any Rocket League workshop map directly into Free Play — no manual file copying, no command-line tools. One click and you are on the map.

Browsing and loading workshop maps from Bakkboard's Workshop panel.

Why workshop maps still matter in 2026

Workshop maps are the best training resource in Rocket League: aerial maps, speed flip maps, wall shot maps, pinch trainers, shot packs, kickoff practice, dribbling courses. They cannot be replicated by official training packs because they use custom environments, physics setups and level geometry.

Loading them manually used to involve copying UPK files into specific game directories and editing config files each time. BakkesMod simplified this. Bakkboard does the same — with a proper UI and without any runtime injection.

The workshop browser

The Workshop tab fetches the full catalog of available maps and displays them in a searchable, sortable list. You can sort by download count, rating, map name, file size or upload date. Each entry shows a thumbnail, the author name, the current rating and the download count.

A search bar filters maps by name or creator in real time. If you already know what you want, typing the first few letters gets you there immediately.

My Maps library

Maps you have already downloaded appear in a My Maps section at the top of the panel. You can load any previously downloaded map with a single click without having to search for it again. If a newer version is available, Bakkboard shows an update indicator.

Loading a map — step by step

  1. Open Bakkboard and go to the Workshop tab.
  2. Browse or search for the map you want.
  3. Click Load. Bakkboard copies the map file to the right location in your Rocket League directory.
  4. In Rocket League, go to Play → Free Play. The workshop map will be loaded as your current arena.
  5. To return to a normal arena, click Unload in Bakkboard and relaunch Free Play.

Steam and Epic Games

Bakkboard detects your Rocket League installation automatically — whether you use Steam or Epic Games. No manual path configuration is needed in most cases. If the game is installed in a custom location, you can point Bakkboard to the correct folder from the Settings panel.
