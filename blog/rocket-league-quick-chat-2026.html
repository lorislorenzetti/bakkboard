Rocket League Custom Quick Chat — Send Any Message Instantly

Rocket League's built-in quick chat gives you a fixed list of preset phrases. Bakkboard's Quick Chat feature breaks that limit: add any text you want, bind it to any key or controller button, and send it instantly during a match.

Configuring custom quick chat messages and key bindings in Bakkboard.

What custom quick chat does

With Bakkboard's Quick Chat tab you can create your own chat shortcuts that work exactly like the built-in quick chat — but with any text you choose. Each entry has three configurable parts:

  • Message text — any string, any length.
  • Trigger — a keyboard key, a modifier combination, or a controller button.
  • Chat scope — Global (all players), Team (your teammates only), or Party (your party only).

Press the trigger while Rocket League is in the foreground and the message appears in chat immediately, in the right channel, without you needing to open the chat input or navigate any menu.

How to set it up

  1. Open Bakkboard and go to the Quick Chat tab.
  2. Click Add to create a new entry.
  3. Type your message in the text field.
  4. Click the key cell next to the entry. Bakkboard listens for your next input — press any key or controller button to bind it.
  5. Select the chat scope: Global, Team or Party.
  6. That is it. The entry is saved automatically.

Your quick chat list persists across sessions. You can edit or remove any entry at any time from the same tab.

Multiple messages, multiple bindings

There is no limit on how many custom quick chat entries you can create. Each one has its own independent key binding and scope, so you can have a full set of shortcuts covering different situations — a team callout on one key, an "I got it" message on another, a party-only joke on a third. Controller and keyboard bindings can be mixed freely.

How it works

Bakkboard runs in the background while you play. When it detects your trigger key while Rocket League is the active window, it sends the message through standard input simulation — no injection, no memory access, no DLL hooking. From Rocket League's perspective it receives the same input as any other message typed by the player. Easy Anti-Cheat sees nothing unusual.

Because Bakkboard only acts when Rocket League is in the foreground, there is no risk of accidentally firing a message while you are tabbed out.
